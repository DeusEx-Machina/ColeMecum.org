Hello World

May 1st, 2015

Hey! If you're reading this right now, congratulations! and welcome to my blog. After self teaching myself some basics about web development I have finally taken the step and got my own personal website.

Setting it up

Getting online with my own domain wasn't easy. If you're like me then you know that the most difficult part of any project is getting it set up so the computer will agree with what it is that you are trying to accomplish. Thankfully GitHub Pages is a thing and is free so webhosting is pretty much taken care of! Also purchasing a domain name is not required since GitHub Pages will publish your site to .github.io. Ain't that nifty?

Customizing my site

There is this nasty little rumor that web developing is not real programming. I think they're right. Anyone who has tried implementing their own custom CSS for a site will know that web development is an art (at least the CSS part). I am not an artist. Luckily for people like me there exists frameworks to help make websites and make them look awesome. I am a fan of Bootstrap personally. It has a grid layout which is common these days. What sets Bootstrap apart from other existing CSS frameworks is the aditional themes that are provided via Bootswatch.

Other than that I have been tweaking little things here and there to help make this site how I want to. Also one day I might do something crazy and add some sort of easter egg game or something. It's a work in progress but it is my work in progress... so stay tuned!
